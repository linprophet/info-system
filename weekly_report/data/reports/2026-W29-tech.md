## 📊 本周观察

本周最清晰的跨主题模式是 **"从 prompt 到 loop"的范式转移正在加速**。Anthropic 的 Boris Cherny、OpenAI 的 agent 团队、以及独立 builder Alex Finn 都公开表示已放弃手动 prompt，转而设计 agent 循环（loops）和 harness。与此同时，Kimi K3 和 Inkling 两个大规模开源模型的发布，使得开源阵营在参数规模和性能上同时逼近闭源前沿——但两者都选择了不同的许可证策略（Kimi K3 使用自定义许可证，Inkling 使用 Apache 2.0），这暗示开源模型的分化可能从"能否用"转向"怎么用"。
