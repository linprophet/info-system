### 🛠️ Agent & Coding Tools

**为什么本周值得看**: AI Engineer World's Fair 将 agent 工具推到了聚光灯下——从 autoresearch 循环到软件工厂概念，从 Cursor 的企业部署到 Warp 的转型，agent 工具正在从实验走向生产。

- [Autoresearch: The feedback loop behind self-improving agents](https://www.latent.space/p/autoresearch-introspection) — *Latent Space* · Introspection CEO Roland Gavrilescu（前 xAI agent 基础设施负责人）定义 autoresearch 为"外循环"：agent 帮助维护和改进主系统，使用反馈信号、评估和人类输入来随时间进步。这是从 agent harness 到反馈循环的关键转变。
- [How Cursor deploys AI inside the enterprise](https://www.latent.space/p/cursor-forward-deployed-engineers) — *Latent Space* · Cursor 的 Forward Deployed Engineering VP Pauline Brunet 描述 Cursor 的"AI 软件工厂"愿景：FDE 团队与组织合作，在整个软件开发生命周期中实施 agent。挑战：将 agent 采用从个人爱好者扩展到整个企业。
- [Warp CEO Zach Lloyd on why software factories are the next phase of coding](https://www.latent.space/p/software-factories) — *Latent Space* · Warp 从 Rust 终端演变为集成 coding agent 的平台，再到开源核心 CLI 工具。Lloyd 认为软件工厂（agent 驱动的自动化软件开发流水线）是 coding 的下一个阶段。
- [Skill engineering and the case against one-shot AI design](https://www.latent.space/p/skill-engineering-design) — *Latent Space* · Impeccable 创始人 Paul Bakaus 提出"skill engineering"学科：给 coding agent 一个改进界面的词汇表（"更大胆"、"更安静"、"更密集"），而不是要求一次重设计整个网站。核心论点：agent 需要领域知识、上下文和精心定义的设计技能，而不仅仅是指令。
- [Getting started with loops | Claude by Anthropic](https://claude.com/blog/getting-started-with-loops) — *Claude Blog* · Anthropic 官方发布的实用指南：从基于回合的工作流进展到基于目标、基于时间和主动式循环。这是对 AIEWF 上"loops"主题的官方背书。
- [Codex for Solutions Engineers: Making AI Tangible for Customers](https://www.youtube.com/watch?v=08hgAtg-P_8) — *OpenAI* · OpenAI 解决方案工程师演示如何使用 Codex 将客户邮件、行业信号、产品信息和 Trustpilot 评论转化为可交互的 demo——包括网站分析、原型修改和配音演示。
