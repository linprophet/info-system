- **[Chat2Workflow: A Benchmark for Generating Executable Visual Workflows with Natural Language](https://huggingface.co/papers/2604.19667)** — _HF Papers · score 0.88_

- **[SkillFlow:Benchmarking Lifelong Skill Discovery and Evolution for Autonomous Agents](https://huggingface.co/papers/2604.17308)** — _HF Papers · score 0.85_

### 🧠 VLM / LLM Post-training

**本周一句话**: VLM/LLM 后训练迎来两大开源里程碑——DeepSeek V4 原生百万上下文与 Qwen3.5-Omni 千亿全模态模型，同时 LLaDA2.0-Uni 用离散扩散统一多模态理解与生成，配套的训练技术、多样性诊断和参数高效扩展亦有多项进展。

- **[Qwen3.5-Omni Technical Report](https://huggingface.co/papers/2604.15804)** — _HF Papers · score 0.95_  
  千亿参数全模态模型，支持 256k 上下文，基于 Hybrid Attention MoE 架构，在 215 项音频与音视频理解、推理、交互子任务中取得 SOTA，关键音频任务超越 Gemini-3.1 Pro。

- **[DeepSeek-V4: a million-token context that agents can actually use](https://huggingface.co/blog/deepseekv4)** — _HF Blog · score 0.85 (also covered in dair.ai)_  
  开源百万上下文模型，Pro (1.6T 总参/49B 活跃) 与 Flash (284B/13B) 原生 1M 上下文训练，混合注意力 CSA+HCA 结合领域专项后训练，大幅拉近开源与 GPT-5.2 的差距。

- **[QIMMA قِمّة ⛰: A Quality-First Arabic LLM Leaderboard](https://huggingface.co/blog/tiiuae/qimma-arabic-leaderboard)** — _HF Blog · score 0.85_  
  新阿拉伯语 LLM 质量优先排行榜，对多个模型在多样化任务上进行系统评测，为该领域提供标准化比较基准。

- **[LLaDA2.0-Uni: Unifying Multimodal Understanding and Generation with Diffusion Large Language Model](https://huggingface.co/papers/2604.20796)** — _HF Papers · score 0.85_  
  用离散扩散 LLM 统一多模态理解与生成，通过 SigLIP-VQ 离散化视觉输入，MoE 主干进行块级掩码扩散，解码器蒸馏实现快速图像重建，性能匹配专业 VLM。

- **[GFT: From Imitation to Reward Fine-Tuning with Unbiased Group Advantages and Dynamic Coefficient Rectification](https://huggingface.co/papers/2604.14258)** — _HF Papers · score 0.70_  
  将 SFT 视作极稀疏策略梯度特例，提出 Group Fine-Tuning，通过无偏组优势学习与动态系数修正，同时解决 SFT 的单路径依赖、熵坍缩和梯度爆炸。

- **[Expert Upcycling: Shifting the Compute-Efficient Frontier of Mixture-of-Experts](https://huggingface.co/papers/2604.19835)** — _HF Papers · score 0.70_  
  提出 expert upcycling 方法：从已有 E 专家模型通过复制与路由扩展构造 mE 专家模型，在继续预训练中渐进扩容，大幅降低大型 MoE 训练成本。

- **[EasyVideoR1: Easier RL for Video Understanding](https://huggingface.co/papers/2604.16893)** — _HF Papers · score 0.65_  
  专门为视频理解优化的 RLVR 框架，解决重复解码开销与超参数敏感性，简化视频模态的强化学习训练流程。

### 🎨 Image Generation

**本周一句话**: 图像生成领域两大主题：一是评估与提升生成结果的身份一致性与空间合理性（StyleID、GSI-Bench），二是将图像生成预训练作为通用视觉理解基础——Vision Banana 证明生成式预训练可涌现强视觉表示，同时阿里 Tstars-Tryon 1.0 落地商业场景。

- **[StyleID: A Perception-Aware Dataset and Metric for Stylization-Agnostic Facial Identity Recognition](https://huggingface.co/papers/2604.21689)** — _HF Papers · score 0.88_  
  提出 StyleID 基准，含感知感知数据集与度量，系统评估风格化肖像的身份保持能力，揭示现有编码器在纹理/几何变化下的严重脆弱性。

- **[Exploring Spatial Intelligence from a Generative Perspective](https://huggingface.co/papers/2604.20570)** — _HF Papers · score 0.88_  
  推出 GSI-Bench，首个通过图像编辑任务量化生成式空间智能的基准，测量模型对 3D 空间约束的遵守与操控能力。

- **[Tstars-Tryon 1.0: Robust and Realistic Virtual Try-On for Diverse Fashion Items](https://huggingface.co/papers/2604.19748)** — _HF Papers · score 0.80_  
  阿里商业级虚拟试穿系统，在极端姿态、光照变化、运动模糊等野外条件下保持高成功率，支持最多 6 张参考图像的多图组合。

- **[Image Generators are Generalist Vision Learners](https://huggingface.co/papers/2604.20329)** — _HF Papers · score 0.75_  
  证明图像生成预训练赋予模型强大的通用视觉理解能力，类比 LLM 的涌现推理；基于 Nano Banana Pro 指令微调的 Vision Banana 在多种视觉任务上取得零样本先进性能。

- **[Elucidating the SNR-t Bias of Diffusion Probabilistic Models](https://huggingface.co/papers/2604.16044)** — _HF Papers · score 0.65_  
  揭示扩散模型中训练与推理阶段信噪比-时间步的对齐偏差，并提出差分校正方法缓解误差累积，提升生成质量。

- **[(1D) Ordered Tokens Enable Efficient Test-Time Search](https://huggingface.co/papers/2604.15453)** — _HF Papers · score 0.65_  
  利用粗-细结构的 1D 有序 tokenizer 替代 2D 网格，使中间态携带语义信息，从而用验证器在测试时进行高效搜索以提升生成控制。

- **[Extending One-Step Image Generation from Class Labels to Text via Discriminative Text Representation](https://huggingface.co/papers/2604.18168)** — _HF Papers · score 0.60_  
  将 MeanFlow 单步生成从类别标签扩展到文本条件，设计判别式文本表示策略，克服集成 LLM 文本编码器时出现的性能退化。

### 🎯 Agent RL & Reasoning

**本周一句话**: RLVR 微调路线持续精细化：本周工作从离策略轨迹利用、困惑度空间解耦到测试时训练的交替校准，均试图突破传统方法过早饱和与多样性坍缩的瓶颈，并用穷尽剪枝提升并行推理效率。

- **[TEMPO: Scaling Test-time Training for Large Reasoning Models](https://huggingface.co/papers/2604.19295)** — _HF Papers · score 0.82_  
  提出 TEMPO，通过交替进行策略优化与 critic 重校准（基于 EM 算法），解决测试时训练的性能平缓和多样性坍缩，使大规模推理模型用更少的额外计算获得更大增益。

- **[Near-Future Policy Optimization](https://huggingface.co/papers/2604.20733)** — _HF Papers · score 0.75_  
  利用 RLVR 中近端高质量离策略轨迹加速收敛，提出 Near-Future Policy Optimization，在同时满足“足够强”与“足够近”条件下最大化有效学习信号。

- **[DiPO: Disentangled Perplexity Policy Optimization for Fine-grained Exploration-Exploitation Trade-Off](https://huggingface.co/papers/2604.13902)** — _HF Papers · score 0.70_  
  通过解耦困惑度空间将样本划分为高困惑度（探索）与低困惑度（利用）子空间，并设计双向奖励分配机制，精细平衡 RLVR 中的探索-利用。

- **[When Can LLMs Learn to Reason with Weak Supervision?](https://huggingface.co/papers/2604.18574)** — _HF Papers · score 0.65_  
  系统实验在稀缺数据、噪声奖励、自监督代理三种弱监督下，RLVR 成功的条件：泛化受训练奖励饱和动力学支配，缓慢预饱和期是关键。

- **[Cut Your Losses! Learning to Prune Paths Early for Efficient Parallel Reasoning](https://huggingface.co/papers/2604.16029)** — _HF Papers · score 0.65_  
  提出 STOP 方法，用可学习 super token 在并行推理早期剪枝无效路径，并首次给出基于信号源（内部/外部）和可学习性的剪枝策略分类体系。

### 🤖 VLA & Embodied

**本周一句话**: 具身智能本周围绕数据稀缺与实时推理展开：UniT 通过视觉锚定的统一动作分词器冲破人-人形机器人壁垒，DeVI 则用合成视频模仿学习实现灵巧操作，OneVL 在自动驾驶中尝试单步潜变量推理以减少延迟。

- **[UniT: Toward a Unified Physical Language for Human-to-Humanoid Policy Learning and World Modeling](https://huggingface.co/papers/2604.19734)** — _HF Papers · score 0.80_  
  提出 UniT，通过视觉锚定的三叉跨重建（动作预测视觉、视觉重构建动作）学习统一潜在动作分词器，实现人类到人形机器人的跨具身策略迁移。

- **[OneVL: One-Step Latent Reasoning and Planning with Vision-Language Explanation](https://huggingface.co/papers/2604.18486)** — _HF Papers · score 0.75_  
  将 VLA 与 World Model 结合，用紧凑潜在 token 替代显式链式思维推理，通过双辅助解码器解释视觉语言，在自动驾驶中同时保持推理质量和实时性。

- **[DeVI: Physics-based Dexterous Human-Object Interaction via Synthetic Video Imitation](https://huggingface.co/papers/2604.20841)** — _HF Papers · score 0.75_  
  利用文本条件合成视频生成人手-物体交互序列，再通过物理仿真模仿学习，实现对新物体的灵巧机器人操控，克服 2D 视频的物理不精确问题。

- **[ClawEnvKit: Automatic Environment Generation for Claw-Like Agents](https://huggingface.co/papers/2604.18543)** — _HF Papers · score 0.60_  
  从自然语言自动生成爪式（夹持型）机器人多样化仿真环境的流水线（解析-生成-验证），用于训练与评测，节省人工环境构建成本。

### 🌍 World Models

**本周一句话**: 世界模型本周向标准化迈出关键一步：WorldMark 推出首个统一评测基准，终结了各模型自建私有场景的碎片化；CityRAG 利用地理锚定数据实现真实城市环境的可导航生成，MultiWorld 则将世界模型拓展至多智能体多视角。

- **[WorldMark: A Unified Benchmark Suite for Interactive Video World Models](https://huggingface.co/papers/2604.21686)** — _HF Papers · score 0.90_  
  第一个交互式视频 world model 统一基准，提供标准化场景、统一动作映射层和一致评价接口，使不同模型可首次公平比较轨迹误差与感知质量。

- **[CityRAG: Stepping Into a City via Spatially-Grounded Video Generation](https://huggingface.co/papers/2604.19741)** — _HF Papers · score 0.80_  
  利用海量地理配准数据作为上下文，训练视频生成模型以重建具有 3D 一致性、可导航的真实城市环境，支持天气变化和动态物体，面向自动驾驶与机器人仿真。

- **[MultiWorld: Scalable Multi-Agent Multi-View Video World Models](https://huggingface.co/papers/2604.18564)** — _HF Papers · score 0.75_  
  多智能体多视角 video world model，通过 Multi-Agent Condition Module 和 Global State Encoder 实现多智能体精确控制，同时保持跨视角观测一致性。

### 🎬 Video Generation

**本周一句话**: 视频生成在数据效率上有亮眼突破：Motif-Video 2B 仅用 10M 片段即达高质量，关键是通过架构解耦分离提示对齐、时序一致与细节恢复；CoInteract 则专注于人-物交互的物理合理性。

- **[Motif-Video 2B: Technical Report](https://huggingface.co/papers/2604.16503)** — _HF Papers · score 0.85_  
  在小于 10M 片段和 100k H200 GPU 小时的极低预算下，通过 Shared Cross-Attention 强化文本控制、结构化解耦提示对齐与时序一致性，实现高质量文本到视频生成。

- **[CoInteract: Physically-Consistent Human-Object Interaction Video Synthesis via Spatially-Structured Co-Generation](https://huggingface.co/papers/2604.19636)** — _HF Papers · score 0.75_  
  端到端人-物交互视频合成框架，引入 Human-Aware MoE 对敏感区域（手、脸）进行专家路由，并利用空间结构化的协同生成保证物理合理的接触，应用于电商与数字营销。

- **[Seeing Fast and Slow: Learning the Flow of Time in Videos](https://huggingface.co/papers/2604.21931)** — _HF Papers · score 0.65_  
  自监督学习视频播放速度检测与估计，并基于此构建大规模慢动作数据集，进而赋能视频生成模型学会控制时间流速，产出不同速度的连贯视频。
