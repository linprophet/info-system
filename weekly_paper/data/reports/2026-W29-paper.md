### 🎯 Agent RL & Reasoning

**本周一句话**: 强化学习后训练进入"规模竞赛"——从万亿参数 zero RL 到百万 token 上下文 RL，同时蒸馏方法（OPD、SEED、TOP-D）成为降低训练成本的关键路径。

- **[LongStraw: Long-Context RL Beyond 2M Tokens under a Fixed GPU Budget](https://huggingface.co/papers/2607.14952)** — _HF Papers · score 0.85_
  提出架构感知的百万 token RL 后训练栈：共享 prompt 无梯度、仅保留模型特定状态、短响应分支逐条回放，在固定 GPU 预算下实现 2M+ token GRPO。
- **[Ring-Zero: Scaling Zero RL to a Trillion Parameters for Emergent Reasoning](https://huggingface.co/papers/2607.12395)** — _HF Papers · score 0.85_
  首次将 zero RL（无人类标注的可验证奖励 RL）扩展到万亿参数，发现可读性差、token 冗余和自适应推理深度不足等规模效应问题，并提出稳定训练管线。
- **[AdvancedMathBench: A Benchmark Suite for Advanced Mathematical Proof Generation and Verification](https://huggingface.co/papers/2607.11849)** — _HF Papers · score 0.85_
  296 道本科到博士资格考试级别的数学证明基准，配备自动验证器进行细粒度过程评估，填补高级数学推理评估空白。
- **[Weak-to-Strong Generalization via Direct On-Policy Distillation](https://huggingface.co/papers/2607.05394)** — _HF Papers · score 0.8_
  提出 Direct-OPD：在小模型上运行 RL 后，将 RL 诱导的策略偏移（而非最终策略）蒸馏到大模型，实现弱到强的泛化。
- **[SEED: Self-Evolving On-Policy Distillation for Agentic Reinforcement Learning](https://huggingface.co/papers/2607.14777)** — _HF Papers · score 0.8_
  自进化框架：将完成的 on-policy 轨迹转化为 hindsight 技能（自然语言描述的可复用工作流），再蒸馏回策略模型，填补稀疏奖励的监督间隙。
- **[Trust Region Policy Distillation](https://huggingface.co/papers/2607.04751)** — _HF Papers · score 0.65_
  提出 TOP-D，通过动态构建近端教师将高方差 OPD 转化为稳定训练范式，提供单调改进界和全局收敛分析。
- **[Demystifying On-Policy Distillation: Roles, Pathologies, and Regulations](https://huggingface.co/papers/2607.13399)** — _HF Papers · score 0.65_
  系统研究 OPD 训练动力学：OPD 本质是探索催化剂（提供密集 token 级引导），其效果完全取决于引导信号质量，揭示师生不匹配和信号退化两种病理。
- **[Spectral Rewiring for Exploration, Purification, and Model Merging](https://huggingface.co/papers/2607.03065)** — _HF Papers · score 0.6_
  提出子空间对齐重连（SAR），保留 RL 更新的频谱核心（推理有效成分）并移除正交分量，实现模型合并时的推理增益保持。

### 🎨 Image Generation

**本周一句话**: 图像生成领域出现"统一化"趋势——理解与生成融合（SC-CMJP、Boogu-Image），同时知识边界（SearchGen）和奖励建模（SpectraReward）成为新关注点。

- **[Boogu-Image-0.1: Boosting Open-Source Unified Multimodal Understanding and Generation](https://huggingface.co/papers/2607.13125)** — _HF Papers · score 0.85_
  开源统一多模态理解与生成模型家族（Base/Turbo/Edit/Edit-Turbo），通过 agentic 推理时缩放提升生成和编辑性能，支持中英双语文本渲染。
- **[Search Beyond What Can Be Taught: Evolving the Knowledge Boundary in Agentic Visual Generation](https://huggingface.co/papers/2607.05382)** — _HF Papers · score 0.85_
  构建 SearchGen-20K 基准和 SearchGen-Corpus-1M 语料库，揭示前沿开放生成器在长尾知识上的 40 分性能崩塌（21-28/100），提出搜索增强 agentic 生成方案。
- **[Read It Back: Pretrained MLLMs Are Zero-Shot Reward Models for Text-to-Image Generation](https://huggingface.co/papers/2607.11886)** — _HF Papers · score 0.75_
  SpectraReward：无需训练的奖励函数，通过测量生成图像中恢复原始 prompt 的对数似然作为奖励，直接复用 MLLM 的图文对齐能力。
- **[Fine-tune video and image models at scale with NVIDIA NeMo Automodel and 🤗 Diffusers](https://huggingface.co/blog/nvidia/scale-diffusers-finetuning-nemo-automodel)** — _HuggingFace · TRL · Blog · score 0.75_
  NVIDIA 与 HuggingFace 合作，提供使用 NeMo Automodel 大规模微调视频和图像扩散模型的方案。
- **[Concurrent Image Understanding and Generation: Self-Correcting Coupled Markov Jump Processes](https://huggingface.co/papers/2607.13188)** — _HF Papers · score 0.7_
  Google 提出 SC-CMJP，在 Masked Diffusion Models 中实现图文交叉模态的自校正耦合生成，一个模态的置信度影响另一模态的转移率。
- **[MetaView: Monocular Novel View Synthesis with Scale-Aware Implicit Geometry Priors](https://huggingface.co/papers/2607.12000)** — _HF Papers · score 0.65_
  快手 Kolors 提出 MetaView，结合隐式几何建模与最小显式 3D 线索，实现单图大视角变化的新视角合成。
- **[Registers Matter for Pixel-Space Diffusion Transformers](https://huggingface.co/papers/2605.16147)** — _HF Papers · score 0.65_
  Yandex Research 发现：像素空间 DiT 虽无 patch-token 异常值，但 register token 仍能改善高噪声水平下的特征图质量，且效果优于潜在空间 DiT。
- **[From RGB Generation to Dense Field Readout: Pixel-Space Dense Prediction with Text-to-Image Models](https://huggingface.co/papers/2607.06553)** — _HF Papers · score 0.55_
  论证从 T2I 模型进行密集预测应使用像素正确读取而非生成式解码，减少不必要的 RGB 生成开销。
