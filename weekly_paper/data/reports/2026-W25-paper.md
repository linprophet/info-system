### 🌍 World Models

**本周一句话**: 世界模型领域迎来密集发布——Kairos、Qwen-RobotWorld、LoopWM、DreamX-World 同时出现，标志着从"视频生成"到"可交互、可预测、可控制的物理世界基础设施"的范式转变。

- **[Kairos: A Native World Model Stack for Physical AI](https://huggingface.co/papers/2606.16533)** — _HF Papers · score 0.9_
  提出原生世界模型栈，包含跨具身数据课程（开放视频+人类行为+机器人交互）、Native Unified Architecture（Hybrid Linear Temporal Attention）和持久状态管理，旨在成为 Physical AI 的操作系统级基础设施。
- **[Qwen-RobotWorld Technical Report: Unifying Embodied World Modeling through Language-Conditioned Video Generation](https://huggingface.co/papers/2606.17030)** — _HF Papers · score 0.9_
  以自然语言为统一动作接口，通过 60 层双流 MMDiT 耦合 Qwen2.5-VL 语义与视频 VAE 潜变量，在机器人操作、自动驾驶、室内导航和人类到机器人迁移四个领域预测未来视觉轨迹。
- **[Looped World Models](https://huggingface.co/papers/2606.18208)** — _HF Papers · score 0.85_
  首个循环架构世界模型，通过参数共享 transformer 块迭代精炼环境隐状态，实现 100x 参数效率，自适应计算深度随预测复杂度自动调整，开辟了模型大小和训练数据之外的第三个 scaling 轴。
- **[DreamX-World 1.0: A General-Purpose Interactive World Model](https://huggingface.co/papers/2606.16993)** — _HF Papers · score 0.85_
  通用交互式文/图到视频世界模型，支持相机导航、区域重访和可提示事件。引入 E-PRoPE 轻量级投影位置编码，通过因果强制、DMD 蒸馏和长 rollout 训练将双向视频生成器转为自回归世界模型。
- **[MolmoMotion: Forecasting Point Trajectories in 3D with Language Instruction](https://huggingface.co/papers/2606.18558)** — _HF Papers · score 0.7_
  提出目标条件 3D 点运动预测任务：给定视觉历史、3D 查询点和语言目标描述，预测未来 3D 轨迹。配套 MolmoMotion-1M 数据集含 116 万段标注。

### 🤖 VLA & Embodied

**本周一句话**: VLA 领域出现两个关键趋势——利用 3D 几何基础模型直接作为策略骨干（GAM），以及通过人类第一人称视频扩展机器人数据（ACE-EGO-0）。

- **[Geometric Action Model for Robot Policy Learning](https://huggingface.co/papers/2606.17046)** — _HF Papers · score 0.8_
  ETH Zurich 提出 GAM，直接复用预训练几何基础模型作为感知、时间预测和动作解码的共享基板，在中间层拆分：浅层为观察编码器，深层联合预测未来观察和输出动作。
- **[ACE-Ego-0: Unifying Egocentric Human and Robotic Data for VLA Pretraining](https://huggingface.co/papers/2606.17200)** — _HF Papers · score 0.8_
  构建可扩展的第一人称视频到动作流水线，将人类视频转换为机器人格式伪动作轨迹，通过统一 token 空间和跨域对齐实现异构数据联合训练。
- **[Playful Agentic Robot Learning](https://huggingface.co/papers/2606.19419)** — _HF Papers · score 0.75_
  UC Berkeley 提出 RATs，具身编码 agent 通过自我导向的玩耍阶段持续获取技能：自主提出可学探索任务、执行机器人代码策略、验证进度、诊断失败、将成功执行蒸馏为持久代码技能库。
- **[Guava: An Effective and Universal Harness for Embodied Manipulation](https://huggingface.co/papers/2606.18363)** — _HF Papers · score 0.75_
  通过系统探索 agent 工作流、动作空间和观察空间的设计空间，识别出三个关键要素：迭代感知-规划-执行循环、结构化 3D 观察表示和层次化动作空间。
- **[From the Hugging Face Hub to robot hardware with Strands Agents and LeRobot](https://huggingface.co/blog/amazon/strands-lerobot-hub-to-hardware)** — _HF Blog · score 0.75_
  Amazon 与 Hugging Face 合作，打通从 Hub 模型到真实机器人硬件的部署流水线。
- **[DragMesh-2: Physically Plausible Dexterous Hand-Object Interaction with Articulated Objects](https://huggingface.co/papers/2606.15133)** — _HF Papers · score 0.7_
  处理铰接物体操作中目标部件无法直接驱动的问题，通过持续物理手-手柄接触产生运动，引入接触感知对抗扰动增强策略鲁棒性。
