## 🔥 30 秒 TL;DR

- **Tuna-2 统一多模态模型**：Meta AI 完全摒弃视觉编码器，直接用像素嵌入实现原生统一理解与生成，单模型在多模态基准上达到 SOTA，证明像素空间可与潜空间方案竞争。([HF Papers](https://huggingface.co/papers/2604.24763))

- **GLM-5V-Turbo 技术报告**：智谱发布面向多模态 Agent 的原生基础模型，将视觉感知作为推理、规划与工具使用的核心组件而非附属接口，在多 agent 评测中取得强力表现。([HF Papers](https://huggingface.co/papers/2604.26752))

- **Nemotron 3 Nano Omni 开源**：NVIDIA 发布首个原生支持音频的多模态模型，在长视频理解、文档智能和 Agent 任务上领先，并采用创新 token 压缩技术降低推理延迟，模型权重开放。([HF Papers](https://huggingface.co/papers/2604.24954), [NVIDIA Blog](https://huggingface.co/blog/nvidia/nemotron-3-nano-omni-multimodal-intelligence))

- **World-R1 视频生成框架**：用 Flow-GRPO 强化学习对齐视频生成与 3D 物理约束，通过预训练 3D 模型反馈强化几何一致性，不改动架构即改善视频几何错误。([HF Papers](https://huggingface.co/papers/2604.24764))

- **Step-Audio-R1.5 技术报告**：阶跃星辰探索大音频语言模型的思维链推理，通过 RLVR 最大化可验证奖励，同时识别“可验证奖励陷阱”——模型可能将连续听觉语境压缩为离散标签而丢失音频智能。([HF Papers](https://huggingface.co/papers/2604.25719))

- **CoPD 协同演进策略蒸馏**：提出 Co-Evolving Policy Distillation，在专家 RL 训练中同步引入学生蒸馏，互教互学以减少多技能合并时的能力损失。([HF Papers](https://huggingface.co/papers/2604.27083))

- **ExoActor 具身控制新范式**：利用大规模第三人称视频生成模型作为统一接口，从任务指令直接合成可行执行过程，无需配对动作标签实现通用人形机器人控制。([HF Papers](https://huggingface.co/papers/2604.27711))

## 📂 按方向 / By Topic

### 🛠️ Agent & Coding Tools

**本周一句话**: 多 agent 评测全面转向动态、多轮、多天的真实工作流环境，其中 Claw 系列基准集中出现，强调环境状态演进、可刷新信号层与可复现快照，而组织化与框架化设计也成新焦点。

- **[ClawMark](https://huggingface.co/papers/2604.23781)** — _HF Papers · score 0.89_  
  A living‑world benchmark for multi‑turn, multi‑day coworker agents. The environment evolves independently between turns (new emails, calendar shifts, knowledge‑base updates), and evidence spans images, PDFs, audio, video, and spreadsheets. The current release includes 100 tasks across 13 professional scenarios with rule‑based verification.

- **[Claw‑Eval‑Live](https://huggingface.co/papers/2604.28139)** — _HF Papers · score 0.88_  
  A live agent benchmark that separates a refreshable signal layer from a reproducible snapshot. Tasks are sourced from public workflow‑demand signals (ClawHub Top‑500 skills) and materialized with fixed fixtures, services, and automated graders, enabling evaluation against evolving real‑world workflow demand.

- **[DV‑World](https://huggingface.co/papers/2604.25914)** — _HF Papers · score 0.87_  
  260 real‑world data‑visualization tasks spanning spreadsheet manipulation, dashboard adaptation across programming paradigms, and proactive intent alignment with a user simulator. Evaluates agent lifecycles beyond code‑sandbox creation‑only tasks.

- **[AutoResearchBench](https://huggingface.co/papers/2604.25256)** — _HF Papers · score 0.85_  
  A benchmark for autonomous scientific literature discovery with complementary Deep Research (multi‑step probing to track down a specific paper) and Wide Research (comprehensively collecting papers that satisfy conditions) tasks.

- **[AgentSearchBench](https://huggingface.co/papers/2604.22436)** — _HF Papers · score 0.85_  
  A large‑scale benchmark built from ~10,000 real‑world agents across providers, formalizing agent search as retrieval and reranking under both executable and non‑executable query settings.

- **Agentic Harness Engineering (AHE)** — _dair.ai · score 0.8_  
  A framework that makes agent‑harness evolution observable and falsifiable via a three‑layer model: revertible components, structured evidence compressed from trajectory tokens, and verifiable decisions. Introduces the idea of harness edits as contracts that can be verified or reverted. ([dair.ai digest 1](https://nlp.elvissaravia.com/p/top-ai-papers-of-the-week-b95), [digest 2](https://nlp.elvissaravia.com/p/ai-agents-weekly-codex-for-everyday))

- **[ClawGym](https://huggingface.co/papers/2604.26904)** — _HF Papers · score 0.8_  
  A full‑lifecycle framework for Claw‑style personal agents, providing 13.5K synthesized training tasks with persona‑driven intents, realistic mock workspaces, and hybrid verification, together with a family of trained ClawGym‑Agent models.

### 🧠 VLM / LLM Post-training

**本周一句话**: 原生统一多模态架构集中爆发——Tuna-2、GLM-5V 均摒弃分离编码器，直接以像素或原生多模态接口处理理解与生成，而 Nemotron 开源则实现了音频-视觉贯通。与此同时，数据效率、幻觉修复、评测成本成为持续关注点。

- **[Tuna-2](https://huggingface.co/papers/2604.24763)** — _HF Papers · score 0.94_  
  Meta AI replaces pretrained vision encoders with simple patch embedding layers, achieving unified visual understanding and generation in a single model. Experiments show SOTA multimodal performance, demonstrating that pixel‑space modeling fully competes with latent‑space approaches.

- **[GLM-5V-Turbo](https://huggingface.co/papers/2604.26752)** — _HF Papers · score 0.93_  
  Technical report for a native multimodal agent foundation model where visual perception is a core reasoning, planning, and tool‑use component, not an auxiliary interface. The report details model design, multimodal training, RL, and toolchain expansion.

- **[Nemotron 3 Nano Omni](https://huggingface.co/papers/2604.24954)** — _HF Papers & NVIDIA Blog · score 0.9_  
  NVIDIA’s first multimodal model with native audio support (plus text, image, video). Delivers leading accuracy in document understanding, long audio‑video comprehension, and computer‑use agents on a 30B‑A3B backbone with token‑reduction techniques for lower latency. Model checkpoints open‑released.

- **[ReVSI](https://huggingface.co/papers/2604.24300)** — _HF Papers · score 0.88_  
  Rebuilds VLM 3D spatial reasoning evaluation by correcting annotation noise and systematic biases from point‑cloud‑derived QA pairs. Introduces frame‑aware scoring to avoid penalizing models for questions unanswerable under sparse frame sampling.

- **[CHAI Video Captioning](https://huggingface.co/papers/2604.21718)** — _HF Papers · score 0.85_  
  A Critique‑based Human‑AI Oversight framework that trains experts to critique and revise model‑generated pre‑captions, producing precise structured captions guided by hundreds of visual primitives developed with filmmakers. Releases open datasets, benchmarks, and recipes for scalable video‑language oversight.

- **[Step-Audio-R1.5](https://huggingface.co/papers/2604.25719)** — _HF Papers · score 0.85_  
  Technical report on training large audio language models with RLVR for chain‑of‑thought reasoning. Identifies the “verifiable reward trap”: while RLVR yields high scores on objective benchmarks, it risks reducing rich acoustic context to isolated text labels, prompting questions about true audio intelligence.

- **[Granite 4.1 LLMs](https://huggingface.co/blog/ibm-granite/granite-4-1)** — _HF TRL Blog · score 0.8_  
  IBM’s deep‑dive into the post‑training methodology and construction of Granite 4.1, offering practical insights into the full LLM building lifecycle.

### 🎬 Video Generation

**本周一句话**: 3D/物理约束与视频生成的结合是本周核心主题——World-R1 用 RL 注入几何一致性，Co-Director 用多 agent 优化语义连贯，Mutual Forcing 则瞄准自回归音视频流的实时生成。

- **[World-R1](https://huggingface.co/papers/2604.24764)** — _HF Papers · score 0.85_  
  Uses Flow‑GRPO reinforcement learning with feedback from pretrained 3D foundation models and VLMs to enforce structural coherence in video generation without architectural changes. Employs periodic decoupled training to balance rigid geometry with dynamic texture.

- **[Mutual Forcing](https://huggingface.co/papers/2604.25819)** — _HF Papers · score 0.75_  
  A framework for fast autoregressive audio‑video generation that couples uni‑modal generators via a causal model trained directly, avoiding multi‑stage distillation. Enables long‑horizon audio‑video synchronization.

- **[Co-Director](https://huggingface.co/papers/2604.24842)** — _HF Papers · score 0.75_  
  A hierarchical multi‑agent framework that casts video storytelling as global optimization: a multi‑armed bandit explores creative directions, while a local multimodal self‑refinement loop mitigates identity drift, balancing exploration and exploitation.

- **[FlowAnchor](https://huggingface.co/papers/2604.22586)** — _HF Papers · score 0.65_  
  A training‑free inversion‑free video editing framework that stabilizes the editing signal by explicitly anchoring both spatial where-to-edit and strength, using spatial‑aware attention refinement and length‑normalized signal scaling.

### 🎨 Image Generation

**本周一句话**: 生成模型后训练首次直接优化 Fréchet 距离 (FD-loss)，一步生成器 FID 达 0.72；同时 RL 被引入图像编辑，通过可验证的思维链取代标量奖励模型，细化编辑控制。

- **[FD-loss](https://huggingface.co/papers/2604.28190)** — _HF Papers · score 0.8_  
  Demonstrates that Fréchet Distance can be optimized representation‑side by decoupling population size for FD from batch size for gradients. A one‑step generator reaches 0.72 FID on ImageNet 256×256, and multi‑step generators are repurposed into strong one‑step models without teacher distillation.

- **[Edit-R1](https://huggingface.co/papers/2604.27505)** — _HF Papers · score 0.78_  
  A verifier‑based RL framework for image editing that uses a chain‑of‑thought reasoning reward model to break instructions into distinct principles, evaluate each, and aggregate scores, moving beyond scalar reward models that ignore fine‑grained instruction alignment.

- **[Meta-CoT](https://huggingface.co/papers/2604.24625)** — _HF Papers · score 0.75_  
  Decomposes any image‑editing operation into a (task, target, required ability) triplet, enabling two‑level decomposition that jointly improves understanding granularity and generalization in unified multimodal models.

- **[Refinement via Regeneration](https://huggingface.co/papers/2604.25636)** — _HF Papers · score 0.65_  
  Proposes regenerating the image to refine misaligned regions instead of editing, enlarging the modification space and avoiding pixel‑level preservation constraints that limit refinement effectiveness.

### 🎯 Agent RL & Reasoning

**本周一句话**: 多技能合并中的既有范式（RLVR + OPD）被发现存在能力损失与行为模式偏差，协同演进蒸馏提供新解法；过程奖励模型开始适配动态 Agent 场景，解决静默错误与过度惩罚探索的问题。

- **[CoPD](https://huggingface.co/papers/2604.27083)** — _HF Papers · score 0.82_  
  Co‑Evolving Policy Distillation interleaves expert RLVR training with bidirectional student distillation, where experts serve as mutual teachers. This reduces the inter‑capability divergence cost and behavioral gap that standard RLVR or post‑hoc OPD pipelines suffer.

- **[DataPRM](https://huggingface.co/papers/2604.24198)** — _HF Papers · score 0.75_  
  An environment‑aware generative process reward model for data‑analysis agents that acts as an active verifier, autonomously interacting with the environment to detect silent errors and avoid wrongly penalizing necessary exploratory actions.

### 🤖 VLA & Embodied

**本周一句话**: 仅一件，但思路极新——将视频生成模型用作任务和场景交互的统一动力学接口，无需传统监督标签。

- **[ExoActor](https://huggingface.co/papers/2604.27711)** — _HF Papers · score 0.8_  
  Leverages large‑scale exocentric video generation as a generalizable control policy for humanoid robots. From a task instruction and scene context, ExoActor synthesizes plausible execution processes that implicitly encode interaction dynamics, circumventing the need for paired action labels.
