- **[ClawGUI](https://huggingface.co/papers/2604.11784)** — _HF Papers · score 0.85_
  首个开源 GUI agent 全栈框架，统一在线 RL 训练（支持并行虚拟环境与真实物理设备）、标准化评估协议和部署，解决了线下评估漂移和闭环训练环境不稳定等瓶颈。

- **[QuanBench+](https://huggingface.co/papers/2604.08570)** — _HF Papers · score 0.85_
  跨 Qiskit/PennyLane/Cirq 三大框架的统一量子代码生成基准，包含 42 个对齐任务，通过可执行功能测试评估 Pass@1、Pass@5，并引入基于 KL 散度的概率输出验收，同时研究反馈修复后的 Pass@1，最高单框架分数达 59.5%。

- **[GameWorld](https://huggingface.co/papers/2604.07429)** — _HF Papers · score 0.85_
  面向多模态游戏 agent 的标准化可验证评估，在浏览器环境中测试直接键盘鼠标控制的 computer-use agent 和通用多模态 agent，克服动作接口异构与启发式验证难题。

- **[DR³-Eval](https://huggingface.co/papers/2604.14683)** — _HF Papers · score 0.85_

### 🧠 VLM / LLM Post-training

**本周一句话**：VL/LLM 后训练方向呈现多元技术报告发布、开放权重化及垂域深耕趋势，从 Nemotron 3 Super 的混合架构到 EXAONE 4.5 的文档 VLM，再到 Audio Flamingo Next 的音频长上下文推理，表明后训练正成为模型能力差异化的主战场。

- **[Nemotron 3 Super](https://huggingface.co/papers/2604.12374)** — _HF Papers · score 0.95_
  120B 参数、12B 激活的混合 Mamba-Attention MoE，首个采用 NVFP4 预训练和 LatentMoE（优化每 FLOP 和每参数精度）的 Nemotron 模型，集成 MTP 层实现原生命题投机解码；经 25T tokens 预训练及 SFT+RL 后训练，支持 1M 上下文，推理吞吐分别提升 2.2× 和 7.5×，在 agentic reasoning 上表现突出。

- **[EXAONE 4.5 Technical Report](https://huggingface.co/papers/2604.08644)** — _HF Papers · score 0.9_
  LG AI Research 首个开放权重 VLM，将专用视觉编码器整合入 EXAONE 4.0 框架，以文档中心的大规模数据做多模态预训练，上下文长度 256K，在文档理解和通用语言能力上均有大幅提升。

- **[Audio Flamingo Next](https://huggingface.co/papers/2604.10905)** — _HF Papers · score 0.9_
  NVIDIA 发布的下一代开放式音频-语言模型，支持最长 30 分钟音频输入，覆盖语音、环境声和音乐；引入 Temporal Audio CoT，将推理步骤显式对齐到时间戳，实现细粒度时序对齐与增强推理。

- **[FORGE](https://huggingface.co/papers/2604.07413)** — _HF Papers · score 0.85_
  面向制造业的细粒度多模态基准，融合真实 2D 图像与 3D 点云，以精确型号等域语义标注，在工件验证、结构表面检测与装配验证 50 个任务上评估 15 个 MLLM，揭示模型在专业语义理解上的显著差距。

- **[Rethinking On-Policy Distillation](https://huggingface.co/papers/2604.13016)** — _HF Papers · score 0.75_
  系统研究 LLM 在线蒸馏动力学，发现成功取决于教师与学生思维模式兼容性，且即使教师得分更高，仍需具备学生从未见过的全新能力；同家族小模型蒸馏时分布上无法区分学生视角，解释了蒸馏失效。

- **[GPT-Rosalind](https://openai.com/index/introducing-gpt-rosalind)** — _OpenAI Blog · score 0.5_
  OpenAI 发布的面向生命科学推理模型，旨在加速药物发现、基因组学分析和蛋白质推理等科研流程。

### 🎯 Agent RL & Reasoning

**本周一句话**：RL for Reasoning 领域同时涌现四项核心算法创新，分别从记忆增强奖励塑造（MEDS）、最小充分知识引导（KnowRL）、预训练空间 RL（PreRL）和序列级 PPO（SPPO）入手，共同解决奖励稀疏、信用分配不稳定和采样多样性下降等瓶颈。

- **[MEDS](https://huggingface.co/papers/2604.11297)** — _HF Papers · score 0.75_
  提出记忆增强动态奖励塑造框架，利用历史 rollout 的模型表示通过密度聚类识别重复错误模式，对高频错误簇施加更重惩罚，在鼓励探索的同时抑制重复错误。

- **[KnowRL](https://huggingface.co/papers/2604.12627)** — _HF Papers · score 0.75_
  将提示设计建模为最小充分知识引导问题：在 RL 训练中将引导分解为原子知识点，通过约束子集搜索（CSS）构造紧凑、交互感知的子集，并识别出剪枝交互悖论，提升困难问题的推理训练。

- **[PreRL](https://huggingface.co/papers/2604.14142)** — _HF Papers · score 0.75_
  直接在预训练空间中用奖励驱动在线更新边缘分布 P(y)，理论验证 log P(y) 与 log P(y|x) 梯度强对齐，弥补 RLVR 仅优化条件分布 P(y|x) 的局限，保留更宽探索能力。

- **[SPPO](https://huggingface.co/papers/2604.08865)** — _HF Papers · score 0.75_
  将推理过程建模为序列级情境老虎机，把整条 CoT 序列视作单一动作，消除值模型的内存开销，在长链推理任务中实现稳定信用分配和更优训练吞吐。

- **[SCOPE](https://huggingface.co/papers/2604.10688)** — _HF Papers · score 0.7_
  提出信号校准在线蒸馏增强：按 rollout 正确性将样本路由至两个互补监督路径，对错误轨迹用教师困惑度加权 KL 蒸馏优先处理教师确实懂得更好的样例，提升 RL 对齐效果。
