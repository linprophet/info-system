- **Mamoda2.5 (ByteDance)**：统一自回归-扩散框架，结合细粒度 MoE 扩散 Transformer (DiT-MoE) 同时处理多模态理解、图像生成和视频生成/编辑，蒸馏版本视频编辑推理加速 95.9 倍。([_alphaxiv_](https://www.alphaxiv.org/abs/2605.02641))
- **MolmoAct2 (AI2/UW)**：全开源 VLA 模型，集成 embodied reasoning VLM 与连续动作专家，在真实 DROID 任务中面对未见物体达到 87.1% 成功率，控制速率提升 2.42 倍。([_alphaxiv_](https://www.alphaxiv.org/abs/2605.02881))
- **HiL-Bench (Scale AI)**：首个系统衡量智能体在信息缺失/模糊时是否懂得主动求助的基准，包含 250 个人工构建的阻断任务，暴露了当前编码评测的盲区。([_HF Papers_](https://huggingface.co/papers/2604.09408))
- **DeepSeek-AI“Thinking with Visual Primitives”**：将点和边界框作为 MLLM 的内在思考单元，弥合复杂视觉推理中的“指代缺口”，显著提升计数、空间推理和视觉 token 效率。([_alphaxiv_](https://www.alphaxiv.org/abs/visual-primitives))

## 📂 按方向 / By Topic

### 🧠 VLM / LLM Post-training

**本周一句话**: 后训练进入“内生对齐”深水区——从模型规范中期训练到 on-policy 蒸馏，多个工作将原则性行为直接注入生成过程；同时 PVF、PVM 等模块试图为长序列视觉推理建立持久感知，而元认知与计数能力的探究则指向 LLM 可靠性的根本缺陷。

- **[Model Spec Midtraining: Improving How Alignment Training Generalizes](https://www.alphaxiv.org/abs/2605.02087)** — _alphaxiv · score 0.9_  
  Anthropic 提出 MSM 阶段：在 pretrain 和标准 alignment 之间插入一个中间训练，教模型理解一份“模型规范”中包含的原则和价值观。该阶段可大幅降低 OOD 场景下的 agentic misalignment，并在低数据条件下将对齐微调的计算效率提升至 60 倍。

- **[On-Policy Distillation](https://www.alphaxiv.org/abs/on-policy-distillation)** — _alphaxiv · score 0.88_  
  Thinking Machines 提出 on-policy 蒸馏：直接用教师模型的稠密 token 级反馈替代稀疏 reward，使 Qwen3-8B 在 AIME'24 上达到 70%，蒸馏成本降至传统 off-policy 的 1/30，同时有效恢复个性化模型丢失的指令跟随能力。

- **[Let ViT Speak: Generative Language-Image Pre-training](https://huggingface.co/papers/2605.00809)** — _HF Papers, alphaxiv · score 0.88_  
  ByteDance 的 GenLIP 用单一 Transformer 同时建模视觉和文本 token，直接让 ViT 以标准语言建模目标预测语言 token。训练仅用 8B 样本，在 14 个多模态 benchmark 上即达到或超越使用 40B 样本的对比基线，展现出极强的可扩展性和简洁性。

- **[Counting as a minimal probe of language model reliability](https://huggingface.co/papers/2605.02028)** — _HF Papers · score 0.88_  
  Stanford 提出“稳定计数容量”检测法，剥离知识和语义，让模型重复计数符号直至出错。在 100+ 模型变体上的结果表明，即使前沿模型，可靠的计数长度也远低于同等推理任务的表现，揭示出知识扩展未能同步提升不确定性意识。

- **[Thinking with Visual Primitives](https://www.alphaxiv.org/abs/visual-primitives)** — _alphaxiv · score 0.85_  
  DeepSeek-AI 将 2D 点和 bounding box 作为 MLLM 的视觉推理原语，使其直接对空间结构进行显式思考，从而缩小复杂推理场景下的“指代缺口”。在计数、空间推理等任务上提升显著，同时减少了所需的视觉 token 总量。

- **[GPT-5.5 Instant System Card](https://openai.com/index/gpt-5-5-instant-system-card)** — _OpenAI Blog · score 0.85_  
  OpenAI 公布了 GPT-5.5 Instant 的安全评估、红队测试和能力基准。该报告是了解最新前沿模型系统设计及对齐实践的窗口，目前尚缺完整技术细节，但系统卡本身已成为安全工程的关键文档。

- **[Persistent Visual Memory: Sustaining Perception for Deep Generation in LVLMs](https://www.alphaxiv.org/abs/2605.00814)** — _alphaxiv, HF Papers · score 0.82_  
  上海 AI 实验室发现自回归 LVLM 在生成长文本时视觉注意力随序列增长而衰减（“视觉信号稀释”），提出 Persistent Visual Memory (PVM) 模块，作为 FFN 的平行分支持续提供距离无关的视觉检索通路。在 Qwen3-VL-8B 上，8 个多模态基准平均提升 4.8%，长输出场景提升 27.3%。

- **[From Context to Skills: Can Language Models Learn from Context Skillfully?](https://huggingface.co/papers/2604.27660)** — _HF Papers · score 0.78_  
  Ctx2Skill 提出推理时技能自动抽取框架：从长文本上下文中自进化式发现、精炼和选择用于解题的程序性技能，无需人工标注或外部反馈，显著改善复杂上下文推理的性能，为“上下文内学习”提供了可复用的技能表示路径。

### 🛠️ Agent & Coding Tools

- **[PhysicianBench: Evaluating LLM Agents in Real-World EHR Environments](https://huggingface.co/papers/2605.02240)** — _HF Papers · score 0.88_  
  斯坦福构建 100 个来自真实临床会诊的长程任务，要求 agent 在 EHR 环境内完成从信息查找到决策建议的复合工作流，通过商用 API 进行可验证执行，弥补了现有医学评测只测知识或单步动作的不足。

- **[AcademiClaw: When Students Set Challenges for AI Agents](https://huggingface.co/papers/2605.02661)** — _HF Papers · score 0.87_  
  80 个由大学生提交并经过专家评审的双语学术任务，覆盖奥赛数学、语言学、GPU 密集型 RL、全栈调试等 25+ 领域，其中 16 个任务需要 CUDA 执行，直接来自“当前 AI agent 无法有效完成”的真实校园挑战。

- **[PatRe: A Full-Stage Office Action and Rebuttal Generation Benchmark for Patent Examination](https://huggingface.co/papers/2605.03571)** — _HF Papers · score 0.85_  
  首次将专利审查全流程（审查意见生成与申请人答复）建模为多轮论证与反驳的动态过程，用 480 个真实案例在 oracle 和检索模拟两种设置下评测 agent 的技术-法律复合推理。

- **[Workspace-Bench 1.0: Benchmarking AI Agents on Workspace Tasks with Large-Scale File Dependencies](https://huggingface.co/papers/2605.03596)** — _HF Papers · score 0.85_  
  涵盖 5 种工作者画像、74 种文件类型、多达 20,476 个文件的真实工作区，设计 388 个任务及对应的依赖图，要求 agent 识别并利用显式与隐式的跨文件依赖关系，填补了工作区层面规模化评测的空白。

- **[Recursive Multi-Agent Systems](https://www.alphaxiv.org/abs/2604.25917)** — _alphaxiv · score 0.8_  
  提出 RecursiveMAS，通过在潜在空间中而非纯文本上进行迭代交互来优化多智能体协作推理，相较基于文本的递归多智能体，准确率平均提升 20.2%，推理速度提高 2.4 倍，token 消耗大幅下降。

### 🎨 Image Generation

**本周一句话**: 自回归生成与流匹配技术齐头并进——字节跳动端到端优化 1D tokenizer 刷新 ImageNet FID 记录，同时 FD-loss 与 PAFM 从损失函数设计入手大幅提升单步/少步生成品质，视觉效果与计算效率的双重突破成为本周画布主色。

- **[End-to-End Autoregressive Image Generation with 1D Semantic Tokenizer](https://huggingface.co/papers/2605.00503)** — _HF Papers · score 0.88_  
  ByteDance Seed 提出端到端联合训练视觉 tokenizer 和自回归生成器，并利用视觉基础模型增强 1D tokenizer，在 ImageNet 256×256 上无引导 FID 达到 1.48，刷新自回归生成 SOTA。

- **[Representation Fréchet Loss for Visual Generation](https://www.alphaxiv.org/abs/2604.28190)** — _alphaxiv · score 0.85_  
  OpenAI 等机构提出 FD-loss，将 FID 直接作为可优化目标解耦为批次级梯度，大幅提升一步生成器输出质量，并能将多步模型高效复用于单步生成，同时引入多表征指标 FDr_k 做更全面评价。

- **[Posterior Augmented Flow Matching](https://www.alphaxiv.org/abs/2605.00825)** — _alphaxiv · score 0.8_  
  华盛顿大学与 Hugging Face 通过引入多目标 posterior 监督改进 Flow Matching，在 ImageNet-1K 上获得最高 3.4 FID50K 的提升，在 CC12M 上也提升 0.92 FID5K，计算开销几乎可忽略。

- **[Linearizing Vision Transformer with Test-Time Training](https://www.alphaxiv.org/abs/2605.02772)** — _alphaxiv · score 0.75_  
  清华提出 T5，在测试时用轻量微调将预训练 Softmax ViT 线性化，恢复近全部性能，同时使 Stable Diffusion 在 2048×2048 分辨率下推理加速 1.47 倍，为扩散骨干网提供了无需重训的加速路径。

- **[ComboStoc: Combinatorial Stochasticity for Diffusion Generative Models](https://huggingface.co/papers/2405.13729)** — _HF Papers · score 0.6_  
  指出扩散模型训练中“组合复杂度”被严重低估，通过构造充分覆盖属性-维度组合空间的随机过程，显著加速网络训练，在图像和 3D 结构化生成任务上均有效。

### 🎯 Agent RL & Reasoning

**本周一句话**: 多轮交互强化学习正从单步 rollout 走向长程稳定控制——T2PO 用不确定性引导探索，延长 horizon 研究找到宏观动作分解作稳定剂，NVIDIA 则将推测解码系统化嵌入 RL 步骤以加速前沿模型训练，代码 RM 的扩展则为多语言多准则奖励提供了新工具。

- **[Themis: Training Robust Multilingual Code Reward Models for Flexible Multi-Criteria Scoring](https://huggingface.co/papers/2605.00754)** — _HF Papers · score 0.9_  
  构建 Themis-CodeRewardBench，涵盖 5 种偏好维度、8 种编程语言，并在其上评测 50+ 现有 RM 后，提出多语言多准则代码奖励模型训练框架，将 RM 应用从功能正确性拓展到更丰富的代码质量标准。

- **[Accelerating RL Post-Training Rollouts via System-Integrated Speculative Decoding](https://www.alphaxiv.org/abs/2604.26779)** — _alphaxiv · score 0.82_  
  NVIDIA 将推测解码内嵌进 RL 训练框架，在 8B 模型上实现 1.41 倍 RL 单步加速，且不损失学习质量；在更大规模 GPU 集群上预估可带来端到端训练 2.5 倍加速。

- **[T²PO: Uncertainty-Guided Exploration Control for Stable Multi-Turn Agentic Reinforcement Learning](https://www.alphaxiv.org/abs/2605.02178)** — _alphaxiv, HF Papers · score 0.78_  
  UCLA 和 Amazon 提出 T²PO，在 token 级别监测不确定性并触发“思考干预”，在 turn 级别抑制低效重复，显著降低 token 消耗与交互轮次，提升 WebShop/ALFWorld 等多轮互动任务的成功率和训练稳定性。

- **[Odysseus: Scaling VLMs to 100+ Turn Decision-Making in Games via Reinforcement Learning](https://huggingface.co/papers/2605.00347)** — _HF Papers · score 0.72_  
  Princeton 用 RL 训练 VLM 在超级马里奥上完成 100+ 回合的长期决策，并引入轻量级 turn-level critic 的 PPO 变体，探索了视觉-动作协同的长程自主学习难题。

- **[On Training Large Language Models for Long-Horizon Tasks: An Empirical Study of Horizon Length](https://www.alphaxiv.org/abs/2605.02572)** — _alphaxiv · score 0.7_  
  来自微软的研究发现任务 horizon 长度是训练 LLM agent 的根本瓶颈，过长 horizon 引发严重不稳定；提出通过宏动作和子目标分解缩短有效 horizon，恢复训练稳定性并泛化到未见过的长任务。

- **[Reinforcement Learning for LLM-based Multi-Agent Systems through Orchestration Traces](https://huggingface.co/papers/2605.02801)** — _HF Papers · score 0.68_  
  提出以编排轨迹（temporal interaction graph）为观测的强化学习框架，将多智能体协作中的 spawn、delegate、aggregate 等事件形式化，设计八族奖励函数，推动了多智能体 RL 的信用分配理论化。

### 🎬 Video Generation

**本周一句话**: 视频生成迈向“统一框架 + 高效缓存”——Mamoda2.5 用单一模型实现了从理解到生成再到编辑的多模态任务，UniVidX 则在扩散先验下将各类生成任务映射到共享空间，MotionCache 首次提出运动感知的逐像素缓存加速。

- **[Mamoda2.5: Enhancing Unified Multimodal Model with DiT-MoE](https://www.alphaxiv.org/abs/2605.02641)** — _alphaxiv · score 0.9_  
  ByteDance 的统一自回归-扩散框架，引入细粒度 MoE 扩散 Transformer，一个模型覆盖多模态理解、图像/视频生成和编辑，在 VBench 2.0 和 OpenVE-Bench 上取得领先，蒸馏版视频编辑推理加速达 95.9 倍。

- **[UniVidX: A Unified Multimodal Framework for Versatile Video Generation via Diffusion Priors](https://www.alphaxiv.org/abs/2605.00658)** — _alphaxiv · score 0.78_  
  BAAI 等机构将各种视觉生成任务统一为共享多模态空间中的条件生成，利用视频扩散模型先验结合架构创新，在跨模态一致性和数据效率上取得突破。

- **[Motion-Aware Caching for Efficient Autoregressive Video Generation](https://huggingface.co/papers/2605.01725)** — _HF Papers · score 0.6_  
  提出 MotionCache，通过帧间差异估计 pixel-level 运动幅度，据此动态决定各像素的去噪步数避免高运动区域的误差累积，加速自回归视频生成的同时保持生成质量。

- **[Map2World: Segment Map Conditioned Text to 3D World Generation](https://huggingface.co/papers/2605.00781)** — _HF Papers · score 0.72_

- **[Generative Modeling with Orbit-Space Particle Flow Matching](https://huggingface.co/papers/2605.02222)** — _HF Papers · score 0.65_

## 📊 本周观察 / Editor's Note

本周一个醒目的跨主题模式是：“生成即对齐”——从 Anthropic 的 Model Spec Midtraining 将模型行为规范编码为生成目标，到 On-Policy Distillation 用教师模型稠密 token 信号取代稀疏奖励，再到多模态 PRISM 在 SFT 与 RL 之间插入 on-policy 蒸馏的分布对齐阶段，多个工作不再将 alignment 视为事后奖惩，而是直接嵌入到模型的生成行为本身。另一个模式是真实场景评测对 agent 能力提出了“求助判断”这一新的元技能要求，HiL-Bench、WindowsWorld 和 PhysicianBench 都在实质上考验智能体在信息不完全或跨应用模糊时何时停止推理、何时请求外部输入，这与 Agent RL 中 T2PO 的“不确定性驱动探索”形成了有趣的镜像——前者衡量，后者训练，本质都是教会模型何时“承认不知道”。
